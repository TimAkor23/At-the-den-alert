## 🎯 Overview

**At the Den** is a comprehensive emergency communication system designed for Lincoln University of Pennsylvania. It enables students to quickly report emergencies, check-in as safe, share their location, and receive real-time broadcasts from administrators. The system also features an AI-powered assistant powered by Google Gemini to provide instant safety guidance.

### Why This Project?

- 🚀 **Real-time emergency response** - Immediate alerts to campus security
- 📍 **Location tracking** - Helps responders find students quickly
- 🤖 **AI-powered assistance** - 24/7 safety guidance using Google Gemini
- 📢 **Centralized communication** - Admins can broadcast to all students instantly
- ✅ **Safety verification** - Students can confirm they are safe during emergencies

## ✨ Features

### 👨‍🎓 Student Features

| Feature | Description |
|---------|-------------|
| **User Registration** | Create a secure student account with email/password |
| **JWT Authentication** | Secure login with JSON Web Tokens |
| **Report Emergency** | Submit emergencies with type, location, and description |
| **Check-In Safe** | Mark yourself as safe during emergencies |
| **Share Location** | Share GPS coordinates with campus security |
| **Emergency Contacts** | Add, edit, and delete personal emergency contacts |
| **Live Broadcasts** | Receive real-time announcements from administrators |
| **AI Chat Assistant** | Ask questions about emergency procedures using Google Gemini AI |
| **Image Upload** | Upload images for AI analysis (fire, injury, hazards) |
| **Voice Input** | Speech-to-text for hands-free AI interaction |

### 👨‍💼 Admin Features

| Feature | Description |
|---------|-------------|
| **Admin Dashboard** | Centralized view of all system activity |
| **View All Emergencies** | Monitor all reported emergencies in real-time |
| **Resolve Emergencies** | Mark emergencies as resolved with notes |
| **View Student Check-Ins** | Track which students have checked in safe |
| **Location Heatmap** | See where students are sharing locations from |
| **Send Broadcasts** | Send announcements to all students instantly |
| **Safety Statistics** | View check-in rates and emergency trends |

### 🤖 AI Assistant Capabilities

| Feature | Description |
|---------|-------------|
| **Emergency Procedures** | Step-by-step guidance for fires, lockdowns, medical emergencies |
| **Image Analysis** | Upload photos of emergencies for AI analysis |
| **Voice Recognition** | Speak your questions instead of typing |
| **General Knowledge** | Answers non-emergency questions too (homework help, etc.) |
| **Concise Responses** | Short, actionable answers (2-3 sentences max) |
